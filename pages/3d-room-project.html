3D Room

This project focuses on my interest in game design by attempting to create a room that I can traverse using standard wsad controls. The keys "A" and "D" rotate the camera view while "W" and "S" control forward and backward motions. Walls also have collision detection to stop moving through them, with the exception of allowing the camera to see through the walls when next to them. The floor is blue and each wall is grey with a square bearing a different color on each wall.
