Cube Physics Test

In addition to exploring collision experiments, I also decided to experiment with a basic physics simulation. In this simulation, a group of 3d objects are inside a rectangular box. There are several controls allowing for moving the objects around. At present, you can select items, increase and decrease current height, move them around using wsad, and deselect them to let them "fall" down. There is basic collision detection allowing objects to collide with other objects. A full list of actionable inputs are listed below.

Controls:

  • Left Click on object: select the object that the cursor is over
  • Left Click on empty space: deselect current object if any
  • Escape: deselect current object if any
  • W key: Move selected object forward
  • S key: Move selected object backward
  • A key: Move selected object left
  • D key: Move selected object right
  • Q key: Rotate selected object
  • E key: Rotate selected object
  • + key: Move selected object up
  • - key: Move selected object down
